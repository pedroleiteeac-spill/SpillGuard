# SpillGuard AI 🏭

**Algoritmo Dinâmico de Atribuição de Custos por Ruptura na Cadeia de Suprimentos**

---

## 📋 Sobre o Projeto
SpillGuard é uma calculadora de custos de paradas de linha (Spill) que quantifica e documenta financeiramente os impactos gerados pela ociosidade de Operadores de Produção (OP) e Operadores de Suporte (OS) decorrentes de falhas de fornecedores.

## 🎯 Objetivo
Desenvolver um MVP funcional que gere Claims (Notas de Débito) auditáveis para recuperação de margem junto a fornecedores.

## 🧠 Metodologia
- Custeio Baseado em Atividades (ABC)
- Framework Scrum + PDCA
- Stack: Python, Streamlit, SQLite

## 📂 Estrutura do Repositório
- `/src` – Código-fonte do projeto
- `/data` – Dados sintéticos da Factory X
- `/docs` – Documentação e referências
- `/tests` – Testes unitários e de integração
- `/notebooks` – Notebooks Jupyter de validação
